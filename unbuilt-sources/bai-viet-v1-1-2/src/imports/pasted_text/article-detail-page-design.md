==================================================
I. BRAND & DESIGN DIRECTION
==================================================

Tone màu thương hiệu:
- White: nền chính
- Black: typography / structure / authority
- Brand Red: CTA / active state / highlight / line / motion accent
- Neutral Gray: metadata / divider / subtle surface

Cảm giác tổng thể:
- premium industrial technology
- global B2B company
- engineered trust
- enterprise-grade
- modern editorial
- studio-level execution
- creative but mature
- expensive looking
- minimal but rich

Tránh:
- cheap startup template
- repetitive card grid
- generic wordpress blog
- ecommerce look
- cluttered layout
- childish effects
- random gradient overload
- too many boxes

- caption
